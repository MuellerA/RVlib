<h1>RVlib</h1>

<p>C++ library for <a href="https://longan.sipeed.com/en/">Sipeed Longan Nano</a>, based on the gd32vf103-sdk framework.</p>

<p>The library handles the most simple use cases for
<ul>
<li>GPIO</li>
<li>USART</li>
<li>SPI (master send)</li>
<li>I2C (master)</li>
<li>CAN</li>
<li>System Tick</li>
<li>OnBoard LEDs</li>
<li>OnBoard LCD</li>
</ul>
</p>
